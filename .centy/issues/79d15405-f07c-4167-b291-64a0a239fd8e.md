# List all centy-\* npm packages

## Overview

We need to discover and list all npm packages published under the `centy-*` naming convention.

## Goal

Query the npm registry to find all packages matching the `centy-*` pattern and compile them into a structured list for review.

## Tasks

- [x] Search npm registry for all packages with the `centy-` prefix
- [x] Document each package with: name, version, description, and download stats
- [x] Identify which packages are official vs community-maintained
- [x] Add the list to the project documentation or a dedicated registry page

## Why

To have a centralized view of the Centy ecosystem on npm, making it easier to:

- Avoid naming conflicts when publishing new packages
- Discover existing integrations and plugins
- Track the growth of the Centy package ecosystem
