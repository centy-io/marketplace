# Display last site generation timestamp on the website

## Description

Show the date/time the website was last generated (i.e. when the static build ran) somewhere visible on the site so visitors know how fresh the data is.

## Acceptance Criteria

* The build timestamp is captured at build time and embedded in the output
* The timestamp is displayed on the website (e.g. footer or a visible UI element)
* The format is human-friendly (e.g. “Last updated: Mar 6, 2026 at 14:30 UTC”)
