# feat: "copy install command" button per package row

## Overview

Add a one-click button to each package row that copies the install command (`npm install <pkg>` or `pnpm add <pkg>`) to the clipboard.

## Tasks

- [x] Add a copy icon button to `PkgRow` (visible on hover)
- [x] Use the Clipboard API (`navigator.clipboard.writeText`) on click
- [x] Show a brief "Copied!" confirmation (tooltip or inline text swap)
- [x] Support both npm and pnpm formats — default to npm, toggle-able per session
- [x] Degrade gracefully in environments where the Clipboard API is unavailable

## Acceptance Criteria

- [x] Clicking the button copies `npm install <pkg>` to clipboard
- [x] Visual confirmation is shown for ~1.5 seconds after copy
- [x] Button is accessible via keyboard and has a descriptive `aria-label`
