# feat: "copy install command" button per package row

## Overview

Add a one-click button to each package row that copies the install command (`npm install <pkg>` or `pnpm add <pkg>`) to the clipboard.

## Tasks

- [ ] Add a copy icon button to `PkgRow` (visible on hover)
- [ ] Use the Clipboard API (`navigator.clipboard.writeText`) on click
- [ ] Show a brief "Copied!" confirmation (tooltip or inline text swap)
- [ ] Support both npm and pnpm formats — default to npm, toggle-able per session
- [ ] Degrade gracefully in environments where the Clipboard API is unavailable

## Acceptance Criteria

- [ ] Clicking the button copies `npm install <pkg>` to clipboard
- [ ] Visual confirmation is shown for ~1.5 seconds after copy
- [ ] Button is accessible via keyboard and has a descriptive `aria-label`
