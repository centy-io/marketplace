# Add favicon to the marketplace app

The marketplace app currently lacks a favicon.

## Source Image

Use the logo from the [`@centy-io/assets`](https://www.npmjs.com/package/@centy-io/assets) package as the source image.

```
pnpm add @centy-io/assets
```

## npm Library

Use the [`favicons`](https://www.npmjs.com/package/favicons) package (v7.2.0) to generate all favicon sizes and formats from the logo.

```
pnpm add -D favicons
```

## Tasks

- Install `@centy-io/assets` and locate the appropriate logo file
- Use `favicons` to generate all favicon assets from the logo
- Add generated favicon files to the `public/` directory
- Configure favicon metadata in `src/app/layout.tsx` using Next.js `Metadata` API
